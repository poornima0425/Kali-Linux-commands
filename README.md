## 📌 What’s Included

This guide covers important command categories:

---

## 🟢 Basic Commands

| Command | Purpose |
|---|---|
| pwd | Show current folder |
| ls | List files and folders |
| cd foldername | Change folder |
| mkdir test | Create new folder |
| rm file.txt | Delete file |
| clear | Clear terminal screen |
| history | Show command history |

---

## 📁 File Commands

| Command | Purpose |
|---|---|
| touch file.txt | Create file |
| cat file.txt | View file content |
| cp file1 file2 | Copy file |
| mv old new | Rename/move file |
| nano file.txt | Edit file |

---

## 🌐 Network Commands

| Command | Purpose |
|---|---|
| ifconfig | Show IP address |
| ping google.com | Test internet |
| netstat -tulnp | Show open ports |
| ip a | Network details |
| traceroute google.com | Trace route |

---

## 👤 User Commands

| Command | Purpose |
|---|---|
| whoami | Current user |
| passwd | Change password |
| sudo su | Become root user |
| adduser name | Create user |

---

## 📦 Package Commands

| Command | Purpose |
|---|---|
| nmap target_ip | Scan target |
| sqlmap -u URL | SQL injection testing |
| hydra | Password testing |
| nikto -h target | Web vulnerability scan |
| msfconsole | Start Metasploit |

| Command | Purpose |
|---|---|
| sudo apt update | Update package list |
| sudo apt upgrade | Upgrade packages |
| sudo apt install nmap | Install tool |
| sudo apt remove toolname | Remove tool |
